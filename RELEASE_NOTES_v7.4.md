# SysGuard v7.4

First public SysGuard release.

## Included

- Baseline-based change detection
- Behavior-aware detection rules
- Process intelligence
- Lightweight file inspection
- Persistence detection
- Threat correlation
- Terminal dashboard
- Desktop GUI
- Web panel
- Native macOS `.app` packaging
- Downloadable `.dmg` installer

## Install

1. Download `SysGuard-7.4-macOS.dmg`
2. Open the disk image
3. Drag `SysGuard.app` into `Applications`
4. Launch the app from `Applications`

## Notes

- The build is ad-hoc signed
- The app is not notarized yet
- On first launch, macOS may ask you to use `Right click > Open`
